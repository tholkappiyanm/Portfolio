
## ✨ Features

- **Modern Dark Theme** with glassmorphism effects
- **Responsive Design** - works on all devices
- **Smooth Animations** - scroll reveals, skill bars, progress indicator
- **Interactive Elements** - 3D tilt effects, hover animations
- **Floating Particles** - animated background
- **Progressive Skill Bars** - animate on scroll
- **Active Navigation** - highlights current section

## 🛠️ Technologies Used

- HTML5
- CSS3 (Variables, Grid, Flexbox, Animations)
- Vanilla JavaScript (Intersection Observer, Event Listeners)
- Google Fonts (Bebas Neue, Plus Jakarta Sans)

## 🚀 Deployment

### GitHub Pages (Free Hosting)

1. Create a new repository on GitHub named `portfolio` (or `yourusername.github.io`)
2. Upload all files to the repository
3. Go to **Settings → Pages**
4. Select **Deploy from a branch** → **main branch** → **/ (root)**
5. Your site will be live at `https://yourusername.github.io/portfolio`

### Custom Domain (Optional)

Add a `CNAME` file with your domain name to use a custom URL.
